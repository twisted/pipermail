mine tends to break the "instanceof" operator in Java rather reliably.

If you don't know what this means, it just means that there's a
twisted.reality.Refrump verb which reloads all the verbs in the game, so
now you have to type "refrump" if you want to test a new verb.

(Note: your verb will first be loaded when it is invoked, not when you
type 'refrump')

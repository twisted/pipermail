The major blocker to be able to make progress on the build infrastructure sensibly is that right now you need to make changes to master on github in order to do development: <https://github.com/twisted-infra/braid/issues/70 <https://github.com/twisted-infra/braid/issues/70>>.  If you could submit some PRs to Braid to make all the service repositories get pushed rather than pulled, we could more easily make a staging or local development environment setup: <https://github.com/twisted-infra/braid/issues/75 <https://github.com/twisted-infra/braid/issues/75>>.

So if you had some time to work on the dev infrastructure that's where to start :).

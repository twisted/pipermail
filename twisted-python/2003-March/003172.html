Look again :)

In fac, the cross-over behaviour is designed to mimic Python's
try/except semantics in a callback oriented way. And exceptions are
*useful*.

 try:
     x = foo(2)
 except FooError:
     x = None

becomes:

 d = foo(2)
 def err(f):
     f.trap(FooError)
     return None
 d.addErrback(err).addCallback(gotX)

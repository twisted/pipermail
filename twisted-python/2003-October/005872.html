IIRC it was quite trivial. Make a class MyDelivery implementing
IMessageDelivery, and then

class MySMTP(smtp.ESMTP):
    delivery = MyDelivery()

factory = protocol.ServerFactory()
factory.protocol = MySMTP
reactor.listenTCP(25, factory)

Here's what it sounds like "returning a Deferred as a result from another
Deferred" describes:

    def f(result):
        return Deferred()

    d = Deferred()
    d.addCallback(f)
    d.callback(None)

This is a useful and very common thing to do.  I see this isn't what you
were attempting to describe now, though.

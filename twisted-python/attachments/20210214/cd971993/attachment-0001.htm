<div dir="ltr">It's time for another Twisted release!<br><br>There are two major announcements for this release:<br><br>- <b>Python 2.7 support has been dropped</b>.  Twisted 21.2.0 supports Python 3.5 and higher only<div>- <b>This will be the last Twisted release to support Python 3.5</b>.</div><div><br></div><div>Twisted 21.2.0rc1 brings the following:<br>- twist dns --pyzone now works on Python 3</div><div>- twisted.web.twcgi now works on Python 3</div><div>- all tests pass on all platforms on Python 3.8, including asyncio tests on Windows</div><div>- tests pass on Python 3.9 (thanks to Michał Górny)</div><div>- type hints, as specified in PEP 484, are now used in functions throughout the codebase</div><div>- mypy (static type checker) is now run over the codebase during CI to help detect problems, and help improve the use of types</div><div>- code in is now formatted with Black (thanks to Tom Most)</div><div>- many CI and automation improvements (thanks to Adi Roiban and Thomas Grainger)</div><div>- many more fixes (225 tickets closed!)<br><br></div><div><br></div><div>You can install by running this command:<br><br>      python -m pip install Twisted==21.2.0rc1<br><br>The full tarball is available at:</div><div>    <a href="https://pypi.org/project/Twisted/21.2.0rc1/#files">https://pypi.org/project/Twisted/21.2.0rc1/#files</a><br><br></div><div>The full NEWS file with all changes is at:</div><div>    <a href="https://github.com/twisted/twisted/blob/twisted-21.2.0rc1/NEWS.rst">https://github.com/twisted/twisted/blob/twisted-21.2.0rc1/NEWS.rst</a><br><br>Please test it, and let me know how your applications fare, good or bad!</div><div>If nothing comes up, 21.2 will release very soon.<br><br>--</div><div>Craig</div></div>

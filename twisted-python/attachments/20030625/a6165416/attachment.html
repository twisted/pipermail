<br>
Resource scripts have been a rather active topic of discussion on the list the<br>
past few days -- you really should check the archives.  You probably don't want<br>
to be using them, as explained by Moshe Zadka in this post:<br>
<br>
    http://twistedmatrix.com/pipermail/twisted-python/2003-June/004693.html <br>
<br>
Regarding form parameters, you'll find them in the .args attribute of a<br>
request.  A trivial example would be:<br>
<br>
    from twisted.web.resource import Resource<br>
    from twisted.web.microdom import escape<br>
    <br>
    class RequestArgResource(Resource):<br>
        def render(self, request):<br>
            return 'request.args: ' + escape(repr(request.args))<br>
<br>

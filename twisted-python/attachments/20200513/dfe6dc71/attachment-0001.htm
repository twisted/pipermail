<div dir="ltr"><div dir="ltr"><div dir="ltr">I would like to propose that Twisted drop support for Python 3.5 one year from now in 2021.<br><br>Reasons:<br><br>1.  Python 3.5.9 will be the final release of Python 3.5, in November 2019 ( <a href="https://www.python.org/dev/peps/pep-0478/">https://www.python.org/dev/peps/pep-0478/</a> )<br>2.  Python 3.6.0 was released in December 2016 ( <a href="https://www.python.org/dev/peps/pep-0494/">https://www.python.org/dev/peps/pep-0494/</a> )<br>3.  PyPy 7.1.1  introduced Python 3.6 support in April 2019 ( <a href="https://morepypy.blogspot.com/2019/04/pypy-711-bug-fix-release.html">https://morepypy.blogspot.com/2019/04/pypy-711-bug-fix-release.html</a> ).<br>    PyPy 7.3.1 is the latest PyPy version ( <a href="https://morepypy.blogspot.com/2020/04/pypy-731-released.html">https://morepypy.blogspot.com/2020/04/pypy-731-released.html</a> ).<br>    <br>4.  Python 3.6 introduced support for variable annotations ( <a href="https://www.python.org/dev/peps/pep-0526/">https://www.python.org/dev/peps/pep-0526/</a> )<br>5.  Python 3.6 introduced new string interpolation (f-string) support ( <a href="https://www.python.org/dev/peps/pep-0498/">https://www.python.org/dev/peps/pep-0498/</a> )<br>6.  Python 3.6 introduced support for asynchronous generators ( <a href="https://www.python.org/dev/peps/pep-0525/">https://www.python.org/dev/peps/pep-0525/</a> ) and asynchronous comprehensions ( <a href="https://www.python.org/dev/peps/pep-0530/">https://www.python.org/dev/peps/pep-0530/</a> )<br><br><br><br>Based on item 2., Python 3.6.0 has been around for almost 4 years, which is a reasonable amount of time.<br>Based on item 3., PyPy supports Python 3.6.<br><br>It might be nice to take advantage of some of the language features introduces in items 4-6.<br><br>I think dropping support one year from now in 2021, would give sufficient notice, and be in line with<br>the Twisted Compatibility Policy ( <a href="https://twistedmatrix.com/documents/current/core/development/policy/compatibility-policy.html">https://twistedmatrix.com/documents/current/core/development/policy/compatibility-policy.html</a> ).<br><br>None of these items are critial, but I thought I would ask, to get people's opinions.<br><br>--</div><div>Craig</div></div></div>

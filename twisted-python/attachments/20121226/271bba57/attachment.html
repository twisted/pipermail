<blockquote class="gmail_quote" style="margin:0 0 0 .8ex;border-left:1px #ccc solid;padding-left:1ex">On behalf of Twisted Matrix Laboratories, I am pleased to announce, in<br>
extremis, the release of Twisted 12.3.<br>
<br>
161 tickets are closed by this release, with the following highlights:<br>
<br>
 * Support for local parallel testing in trial<br>
<br>
 * A new "react" function to easily run a single asynchronous function<br>
in a script with the reactor.<br>
<br>
 * Partial support for Python 3.3.<br>
<br>
Note that only Python 2.7 is supported on Windows now. We also require<br>
zope.interface 3.6.0 or newer.<br>
<br>
<br>

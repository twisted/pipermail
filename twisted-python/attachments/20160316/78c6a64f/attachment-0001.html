> On behalf of Twisted Matrix Laboratories, I am honoured to announce the release of Twisted 16.0!<br>
><br>
> Twisted 16.0 brings some important changes, and some nice-to-haves as well. The major things are:<br>
><br>
> - TLS endpoints have arrived! They're like the old `ssl:` endpoints, but support faster IPv4/IPv6 connections (using HostnameEndpoint) and always do hostname verification.<br>
> - Conch now uses Cryptography instead of PyCrypto for underlying cryptographic operations. This means it'll work much better on PyPy!<br>
> - Headers objects (notably used by t.web.server.Request) now support Unicode for the vast majority of cases, encoding keys to ISO-8859-1 and values to UTF-8.<br>
> - WSGI support and AMP have been ported to Python 3, along with a handful of other modules.<br>
> - More shedding of the past, with the GTK+ 1 reactor being removed.<br>
> - Over 45 tickets have been closed since 15.5.<br>
><br>
> For more information, check the NEWS file (link provided below).<br>
><br>
> You can find the downloads at <<a href="https://pypi.python.org/pypi/Twisted" rel="noreferrer" target="_blank">https://pypi.python.org/pypi/Twisted</a>> (or alternatively <<a href="http://twistedmatrix.com/trac/wiki/Downloads" rel="noreferrer" target="_blank">http://twistedmatrix.com/trac/wiki/Downloads</a>>). The NEWS file is also available at <<a href="https://github.com/twisted/twisted/blob/twisted-16.0.0/NEWS" rel="noreferrer" target="_blank">https://github.com/twisted/twisted/blob/twisted-16.0.0/NEWS</a>>.<br>

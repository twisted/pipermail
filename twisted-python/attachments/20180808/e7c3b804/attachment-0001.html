<div dir="ltr">Hello all,<br><br>I'm pleased to announce a new release of txkube, a Twisted-based library for interacting with Kubernetes using the HTTP API.  The big news for this release is support for Python 3.6.  Also included is support for multiple configuration files in the <font face="monospace, monospace">KUBECONFIG</font> environment variable which allows for better configuration management practices.<br><br>Here is an example of txkube usage, taken from the README:<br><br><font face="monospace, monospace">   from __future__ import print_function<br>   from twisted.internet.task import react<br><br>   from txkube import network_kubernetes_from_context<br><br>   @react<br>   def main(reactor):<br>       k8s = network_kubernetes_from_context(reactor, u"minikube")<br>       d = k8s.versioned_client()<br>       d.addCallback(<br>           lambda client: client.list(client.model.v1.Namespace)<br>       )<br>       d.addCallback(print)<br>       return d<br><br></font>You can <a href="https://pypi.python.org/pypi">download txkube from PyPI</a><br>You can <a href="https://github.com/LeastAuthority/txkube">contribute to its development on GitHub</a>.<br><br>Thanks to <a href="https://leastauthority.com/">Least Authority TFA GmbH</a> for sponsoring this development and to Craig Rodrigues for his efforts on Python 3 porting work.<br><br>Jean-Paul Calderone<div><<a href="https://as.ynchrono.us/">https://as.ynchrono.us/</a>><div><br></div></div></div>

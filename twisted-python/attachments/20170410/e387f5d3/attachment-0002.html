<div dir="ltr">Hello all,<div><br></div><div>I'm pleased to announce the release of txAWS 0.3.0.  txAWS is a library for interacting with</div><div>Amazon Web Services (AWS) using Twisted.</div><div><br></div><div>You can download the release from <a href="https://pypi.python.org/pypi/txAWS">PyPI</a> (<a href="https://pypi.python.org/pypi/txAWS">https://pypi.python.org/pypi/txAWS</a>).</div><div><br></div><div>txAWS development uses GitHub for <a href="https://github.com/twisted/txaws">issues and source control</a> (<a href="https://github.com/twisted/txaws">https://github.com/twisted/txaws</a>).</div><div><br></div><div>Since the last release, the following enhancements have been made:</div><div><ul><li>Jean-Paul Calderone added basic Route53 support.</li><li>Mark Williams add Auth v4, now used by the S3 and Route53 support.</li><li>Francisco Souza added support for VPCs to some EC2 APIs.<br></li><li>Drew Smathers added multi-part S3 upload support.<br></li><li>Several people added other S3 client features such as pagination and streaming uploads.</li><li><i>txaws.testing</i> now provides S3 and Route53 support.</li></ul></div><div>Additionally, the following APIs have been deprecated:</div><div><ul><li>The <i>client</i> attribute of <i>txaws.client.base.BaseQuery</i></li><li>The <i>date</i> attribute of <i>txaws.s3.client.Query</i></li></ul>Thanks to everyone who contributed and to <a href="https://leastauthority.com/">Least Authority Enterprises</a> (<<a href="https://leastauthority.com/">https://leastauthority.com/</a>>) for sponsoring my work on this release.</div><div><br></div><div>Jean-Paul Calderone</div><div><br></div></div>

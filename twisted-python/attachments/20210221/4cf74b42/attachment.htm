    <p>Thanks Craig for getting this release ready.</p>
    <p>It looks like it is incompatible with Python 3.5.2: exception as
      below. There's no real reason you should care about this, except
      that it is (still!) what Ubuntu 16.04LTS ships with, and Ubuntu
      claim to still be supporting 16.04 until April this year, so this
      might catch a few people by surprise, as it did me[1]. Anyway, I
      suggest bumping `python_requires` in `setup.cfg` (<a
        class="moz-txt-link-freetext"
        href="https://github.com/twisted/twisted/blob/trunk/setup.cfg#L27"
        moz-do-not-send="true">https://github.com/twisted/twisted/blob/trunk/setup.cfg#L27</a>),
      to stop the new version being installed on systems where it won't
      work.<br>

<br>
Just in time for PyCon US, this release brings a few headlining features (like the haproxy endpoint) and the continuation of the modernisation of the codebase. More Python 3, less deprecated code, what's not to like?<br>
<br>
- twisted.protocols.haproxy.proxyEndpoint, a wrapper endpoint that gives some extra information to the wrapped protocols passed by haproxy;<br>
- Migration of twistd and other twisted.application.app users to the new logging system (twisted.logger);<br>
- Porting of parts of Twisted Names' server to Python 3;<br>
- The removal of the very old MSN client code and the deprecation of the unmaintained ICQ/OSCAR client code;<br>
- More cleanups in Conch in preparation for a Python 3 port and cleanups in HTTP code in preparation for HTTP/2 support;<br>
- Over thirty tickets overall closed since 16.1.<br>
<br>
For more information, check the NEWS file (link provided below).<br>
<br>
You can find the downloads at <<a href="https://pypi.python.org/pypi/Twisted" rel="noreferrer" target="_blank">https://pypi.python.org/pypi/Twisted</a>> (or alternatively <<a href="http://twistedmatrix.com/trac/wiki/Downloads" rel="noreferrer" target="_blank">http://twistedmatrix.com/trac/wiki/Downloads</a>>). The NEWS file is also available at <<a href="https://github.com/twisted/twisted/blob/twisted-16.2.0/NEWS" rel="noreferrer" target="_blank">https://github.com/twisted/twisted/blob/twisted-16.2.0/NEWS</a>>.<br>

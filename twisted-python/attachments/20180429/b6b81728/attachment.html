    <p>On behalf of Twisted Matrix Laboratories, I'm honoured to
      announce the release of Twisted 18.4.0!</p>
    <p>The highlights of the release are:</p>
    <p> </p>
    <div class="de2">- The dropping of Python 3.3 support.</div>
    <div class="de1">- Python 3 fixes (notably to trial -j,
      asyncioreactor, conch, and mail)</div>
    <div class="de1">- Python 3 TCP speed improvements (less copying
      when sending data)</div>
    <div class="de1">- Better TLS curve selection support for both old
      and new OpenSSLs</div>
    <div class="de1">- IPv6 fixes for WSGIResource</div>
    <div class="de2">- 60+ closed tickets with many fixed bugs!</div>

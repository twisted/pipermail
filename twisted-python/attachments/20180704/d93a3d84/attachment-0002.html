<div dir="ltr"><div>Hello everyone, from scenic Heathrow airport. It's time for another Twisted release, and this one has got some goodies. Included in this release are:</div><div><br></div><div>- better support for async/await coroutines in regards to exception and traceback handling;</div><div>- better support for reporting tracebacks in inlineCallbacks, now showing what you would expect in synchronous-like code</div><div>- the epoll reactor now no longer hard-locks when running out of file descriptors</div><div>- directory rendering in t.web works on Python 2 again</div><div>- manhole's colouriser is better at handling Unicode</div><div>- setting the groundwork for Python 3.7 support. Note that Python 3.7 is currently not a supported platform on any operating system, and may completely fail to install, especially on Windows.</div><div><br></div><div>You can get this hot, fresh Twisted for testing through PyPI:</div><div><br></div><div>pip install Twisted==18.7.0rc1</div><div><br></div><div>You can read the full changelog here:</div><div><br></div><div><a href="https://github.com/twisted/twisted/blob/release-18.7.0-9475/NEWS.rst">https://github.com/twisted/twisted/blob/release-18.7.0-9475/NEWS.rst</a></div><div><br></div><div>Please do some testing, and if all goes well, 18.7 will release proper in a week.</div><div><br></div><div>- Amber<br></div><div><br></div></div>

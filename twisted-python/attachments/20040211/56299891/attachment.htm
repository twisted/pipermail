<P><FONT SIZE=2>I think you might be a little confused; Twisted's primary purpose is as an</FONT>
<BR><FONT SIZE=2>application development framework.&nbsp; The installer you downloaded installs</FONT>
<BR><FONT SIZE=2>tools and libraries for developing applications in Python.&nbsp; Although there are some features that can be used standalone (twistd -f web.tap comes to mind), it is not an application unto itself and these tools are secondary to the purpose of Twisted which is as a developers' resource.</FONT></P>

<P><FONT SIZE=2>As such, any minimum disk requirements would be only for the toolkit, not</FONT>
<BR><FONT SIZE=2>including Python (which is required).&nbsp; Minimum processor specs and memory</FONT>
<BR><FONT SIZE=2>requirements are meaningless outside of the context of an application.</FONT>
</P>

<P><FONT SIZE=2>That said:</FONT>
<BR><FONT SIZE=2>* the installer you downloaded requires 65MB of disk space including the</FONT>
<BR><FONT SIZE=2>documentation which takes up most of the space.&nbsp; There is another installer at</FONT>
<BR><FONT SIZE=2>the same site that does not install doco, and requires 12MB installed.</FONT>
<BR><FONT SIZE=2>* Any support you get for any operating systems would be on a contract basis</FONT>
<BR><FONT SIZE=2>with the developers, as this is an open source project.&nbsp; However, Linux, Mac</FONT>
<BR><FONT SIZE=2>OS X 10.2 and 10.3 and Windows 98 and later are all known to work with</FONT>
<BR><FONT SIZE=2>Twisted, as well as various flavors of Unix.&nbsp; Basically, anywhere Python runs</FONT>
<BR><FONT SIZE=2>(<A HREF="http://python.org" TARGET="_blank">http://python.org</A>).</FONT>
</P>

Bringing not only headlining features but also a lot of incremental improvements, this release has got plenty to like:<br>
<br>
- twisted.logger has landed! This is a brand-new, feature-rich logging framework.<br>
- Python 3.4 is now a supported platform for all the Py3 ported modules.<br>
- twisted.trial.unittest.TestCase's assertEqual, assertTrue, and assertFalse methods now pass through the standard library's more informative failure messages.<br>
- twisted.python.filepath.FilePath now supports Unicode (text) paths properly, and includes as{Bytes,Text}Mode methods for interacting with APIs that require a text/bytes-only FilePath.<br>
- twisted.mail.smtp.sendmail now supports ESMTP and provides a high-level interface for sending mail.<br>
- The following parts of Twisted are now ported to Python 3:<br>
    - twisted.internet.process<br>
    - twisted.cred.credentials<br>
    - twisted.python.modules<br>
    - twisted.internet.kqreactor<br>
    - twisted.internet.endpoints.ProcessEndpoint<br>
    - twisted.web.static<br>

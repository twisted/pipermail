You need to specify a Content-Type in your POST request, e.g.:<div class="Ih2E3d"><br>
<br>
        d = client.getPage(self.getURL("test"), method="POST",<br></div>
                           postdata=urllib.urlencode({"test": "data"}),<br>
                           headers={"Content-Type": "application/x-www-form-urlencoded"})<br>
<br>
Otherwise, your POST request is sent as text/html, and the HTTP server code won't know to parse args out of the request body.<br>

</div>What is the advantage of using more?<br></blockquote></div><br>If you use more with a DeferredList you can have [effectively] parallel page fetches.  It's like spawning threads for each page, but twisted doesn't actually spawn threads for this I don't think.  The net effect is pretty much the same since it will send out all the requests before the replies come in and then handle all of them as they come in.  Especially, for a system reading multiple websites, if two or more are down in a DeferredList your system will overlap the timeout periods rather than waiting them out sequentially.<br>

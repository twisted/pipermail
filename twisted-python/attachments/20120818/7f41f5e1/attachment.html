<div>The pre-release for the upcoming Twisted 12.2 is now available!</div><br>You can download the tarballs here:<br><br>    <a href="http://twistedmatrix.com/~ashfall/12.2.0pre1/">http://twistedmatrix.com/~ashfall/12.2.0pre1/</a><br>

<br>Highlights for this release include:<br><br>  * To be able to work on Python3 support, Python 2.5 is no longer supported.<br><br>  * twisted.mail.imap4 now serves BODYSTRUCTURE responses which provide more information and conform to the IMAP4 RFC more closely.<br>

<br>  * twisted.conch now supports commercial SSH implementations which don't comply with the IETF standard.<br><br>  * twisted.internet.endpoints now provides several new endpoints, including a TCP client endpoint that resolves hostnames to IPv6 host addresses.<br>

<br>  * IReactorSocket.adoptStreamConnection, implemented by some reactors, allows adding an existing established connection to the reactor.<br>  <br>For the complete list, see the NEWS file.<br><br>Note that starting with the next release after 12.2, Twisted will begin requiring zope.interface 3.6 (as part of Python3 support).<br>

<br>Please download the tarballs and test them rigorously, so we can have a great release!<br><br><br>Thanks!<br><br>--<br>Ashwini Oruganti (ashfall)

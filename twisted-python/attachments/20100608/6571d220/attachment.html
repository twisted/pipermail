Two options:<div><ol><li>(as Itamar notes) use spawnProcess for all your process spawning needs</li><li>reactor.run(installSignalHandlers = False) and do your own signal handling</li></ol><div>For option #2, you'll almost certainly want to install SIGTERM and SIGINT handlers which call reactor.stop().</div>

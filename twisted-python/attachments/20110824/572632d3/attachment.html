Are you getting the error "Operation not permitted" when you try to do directory listing?<div>This can be caused by running the SFTP server under a user which can't change his</div><div>effective group and uid to 0 (uid of root user).</div>
<div>This is because when you login as user "foo" the server needs to execute all subsequent</div><div>commands as this user (e.g. when you upload a file this file should be owned by the user "foo")</div>
<div>and therefore temporary changing to user "foo".</div><div>This is exactly what the default implementation of UnixConchUser wants to do, but fails and therefore you got </div><div>that error.</div><div><br></div>

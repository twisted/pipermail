Hi Patrick

You have

    try:
        client2 = Client(username = "Test")
        yield client2.connect(port = server_port)
        assert False
    except UsernameAlreadyExists, ex:
        pass

You could instead try:

    client2 = Client(username = "Test")
    d = client2.connect(port = server_port)
    self.failUnlessFailure(d, UsernameAlreadyExists)
    yield d

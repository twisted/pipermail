I think it may just be the built-in Python on Leopard:

On OS X:

In [1]: def f():
   ...:     for i in range(100000):
   ...:         a = 1 + i
In [2]: %timeit f()
100 loops, best of 3: 11.8 ms per loop

Same hardware, but Windows:

In [1]: def f():
   ...:     for i in range(100000):
   ...:         a = 1 + i
In [2]: %timeit f()
100 loops, best of 3: 8.33 ms per loop

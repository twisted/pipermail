the problem was caused by an unicode string. Changing this line:

|                              "attachment; filename=\"" + os.path.basename(video['title'] + "." + video['format']) + "\"")
to this:
||                              "attachment; filename=\"" + str(os.path.basename(video['title']) + "." + video['format']) + "\"")|

fixes it.

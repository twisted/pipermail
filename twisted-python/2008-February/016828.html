>        try:
>            d = client.getPage("http://www.google.com")
>        except Exception:

I guess this should be:

       try:
           d = client.getPage("http://www.google.com")
       except Exception:
           d = defer.fail()

Alternatively:

        d = maybeDeferred(client.getPage, "http://www.google.com")

although that's probably maybeDeferred() abuse.

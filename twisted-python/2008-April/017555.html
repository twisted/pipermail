> I think the easiest way to do logging is
>
> from   twisted.python                              
> import log
>
> ...
>
> log.startLogging(a file descriptor - i.e., sys.stdout)
>
> Twisted will start emitting log messages. You can use
> log.msg() for additional messages.
>

Hi everyone,

Just so you know, Twisted has had two more builders added to the supported list. These are:

- FreeBSD 10.1 with Python 2.7
- Apple OS X 10.10 Yosemite with Python 2.7

Many thanks to Tavendo for the FreeBSD builder and Glyph for updating one of our OS X 10.6 builders to 10.10 so that I could get Python 2.7 builds running on it.

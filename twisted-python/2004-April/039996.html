Maybe by making a parallel to imperitive style we can make it clearer.

d = op()
d.addCallbacks(callback, errback)

is equivilient to

try:
    r = op()
except e:
    return errback(e)
else:
    return callback(r)

d = op()
d.addCallback(callback).addErrback(errback)

is equivilient to

try:
    return callback(op())
except e:
    return errback(e)

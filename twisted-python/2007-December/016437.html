To summarize Jean-Paul's link, you can just a two-liner patch in the
epollreactor.py to fix this (after line 201 if you're using the
twisted 2.50release)

if event & _epoll.HUP or event & _epoll.ERR:
                    why = main.CONNECTION_LOST

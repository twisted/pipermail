The Divmod server experienced a hardware failure.  As there is no longer a dedicated company maintaining this code, efforts to recover that data are ongoing, but sporadic.  The website (wiki) in particular, is gone for the forseeable future, and links to it need to be updated.

However, all of the code (including all in-progress branches) was successfully recovered and migrated from Subversion to Bazaar, and is now available at <https://launchpad.net/divmod.org/>.

The projects are still alive and have several interested maintainers.

Also, the WebDevelopmentWithTwisted page needs updating to mention twisted.web.template.

Vague question deserves a vague answer.

With the very similar setup I have run into spurious errors trying to
write to a Response.
The amount of data and writing to a stream seemed to matter.
For example:

>>> resp = Response(mimetype='text/json')
>>> simplejson.dump(obj, resp)

was failing on occasion, but

>>> Response(simplejson.dumps(obj), mimetype='text/json')

worked.  Besides, like JP wrote, what the traceback says.

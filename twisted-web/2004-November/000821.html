>class Implementation(object):

>        def _cb():
>            print "Done!"

Your _cb is in no way connected to the Deferred.
Try

    def bar(self, baz):
        def _cb(result):
            print "Done!", result
            return "hello..."

        print "baz!", baz

        d = defer.Deferred()
	d.addCallback(_cb)
        reactor.callLater(0, d.callback, 42)
        return d

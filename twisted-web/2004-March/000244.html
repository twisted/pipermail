Hi. Here's a patch that adds Debian packaging to Nevow.

But, before you add it to the CVS repository, I'd like to
ask what are your feelings as to how and who should maintain
nevow.deb?

1) Should it be done pretty much like with Twisted,
with the "official" debian/ dir stored in the upstream CVS.

2) Should you just totally ignore the fact that there is any
Debianization going on, and let someone (me?) use a patch like
the attached one?

3) Should there be an "unofficial" debianization stuff in Nevow
CVS, and a separate "official" version by someone (me?).


Option 1) pretty much requires giving the Debian maintainer CVS commit
access.

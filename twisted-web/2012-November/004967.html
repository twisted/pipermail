As an additional test I took the simple demo from twisted web in 60 seconds
and checked that out.

from twisted.web.server import Site
from twisted.web.static import File
from twisted.internet import reactor

resource = File('/tmp')
resource.indexNames = ['adminuser.html']
factory = Site(resource)
reactor.listenTCP(8888, factory)
reactor.run()

The adminuser.html has the same sort of link to get the css and in this
case all renders successfully.

But here the whole site is set up as static so I can see why that might
have worked.

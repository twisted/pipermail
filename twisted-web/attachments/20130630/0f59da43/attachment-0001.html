<div>On behalf of  Twisted Matrix Laboratories, I am pleased to announce the release of</div><div>Twisted 13.1.</div><div><br></div><div>Highlights for this release include:</div><div><br></div><div>  * trial now has an --exitfirst flag which stops the test run after the first error or failure.</div>

<div><br></div><div>  * twisted.internet.ssl.CertificateOptions now supports chain certificates.</div><div><br></div><div>  * twisted.conch.endpoints.SSHCommandClientEndpoint is a new IStreamClientEndpoint</div><div>    which supports connecting a protocol to the stdio of a command running on a remote host</div>

<div>    via an SSH connection.</div><div><br></div><div>  * twisted.web.xmlrpc.QueryProtocol now generates valid Authorization headers for long user</div><div>    names and passwords.</div><div><br></div><div>  * twisted.internet.endpoints.connectProtocol allows connecting to a client endpoint using only</div>

<div>    a protocol instance, rather than requiring a factory.</div><div><br></div><div>For more information, see the NEWS file here:</div><div>  <a href="http://twistedmatrix.com/Releases/Twisted/13.1/NEWS.txt">http://twistedmatrix.com/Releases/Twisted/13.1/NEWS.txt</a></div>

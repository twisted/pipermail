st.stream), and the multipart boundary from the content-type, then call its read() method repeatedly to get back tuples of (fieldname, filename, ctype, dataStream). Then you use dataStream.read() to get back data blocks and can do whatever you like with them.</DIV></DIV><BR class="khtml-block-placeholder"><DIV>I can help explain any of the alternatives in more detail if you let me know which you're really interested in.</DIV><DIV><BR class="khtml-block-placeholder"></DIV><DIV>James</DIV></BODY></HTML>

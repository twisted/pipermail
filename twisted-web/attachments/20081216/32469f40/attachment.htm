The first thing would be to check that you&#39;re running at least Python 2.5. Previous versions have garbage collection problems on long-running apps like a Twisted web app.<br>
<br>
Athena-based apps should not inherently leak memory, but adding leaks of your own creation isn&#39;t too hard ;-)<br>
<br>
If you are running 2.5 or later, you should try to reduce your app to the smallest possible test version that demonstrates the leak, and someone here can probably be of more help.<br>

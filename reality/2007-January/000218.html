The code is now in the Imaginary directory in Divmod's svn repository.

To check it out, co http://divmod.org/svn/Divmod/trunk/Imaginary

You'll need a bunch of other Divmod projects to run it, though, so you
might want to check out http://divmod.org/svn/Divmod/trunk instead.  Not
everything you'll get is necessary, but a lot of it is. :)

You can also browse this online at http://divmod.org/trac/browser/trunk/

Things have been quiet for a few months, I think we're gathering strength
for some kind of simulation time refactoring/cleanup.
